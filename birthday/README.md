# 张雨欣生日快乐页面使用说明

这是一个为您精心设计的生日庆祝网页，采用了粉色、金色和白色的浪漫色调，并包含了丰富的交互效果。

## 核心功能
- **视觉设计**：温馨的粉色渐变背景，搭配动态飘落的五彩纸屑和上升的气球。
- **互动元素**：
  - **点击蛋糕**：点击蛋糕可以“吹灭”蜡烛，并触发炫彩纸屑爆发和祝福语显示。
  - **音乐控制**：右下角的悬浮按钮可控制背景音乐的播放/暂停。
  - **祝福卡片**：展示了精美的插画和温馨的祝福语，支持响应式布局。
- **响应式适配**：完美兼容手机、平板和电脑浏览器。

## 如何使用
1. **本地运行**：
   - 直接双击文件夹中的 `index.html` 文件，使用任意现代浏览器（推荐 Chrome, Edge, Firefox）打开即可。
2. **背景音乐**：
   - 页面右下角有音乐控制按钮。由于浏览器安全策略，音乐通常需要用户手动点击一次页面后才能播放。

## 如何自定义内容
您可以轻松地修改网页中的文字或图片：

1. **修改姓名**：
   - 打开 `index.html`，搜索 `const CONFIG = { name: "张雨欣", ... }`。
   - 将 `"张雨欣"` 修改为您想要的姓名。
2. **修改祝福语**：
   - 找到 `<main class="cards-container">` 标签内的 `<p>` 标签，直接修改文字内容。
3. **更换图片**：
   - 找到 `<div class="card">` 内部的 `<img>` 标签，将 `src` 属性替换为您自己的图片路径或网络链接。
4. **更换背景音乐**：
   - 找到 `<audio id="bg-music">` 标签，将 `src` 属性替换为您的音乐文件路径或链接。

## 技术规范
- 纯 HTML5 / CSS3 / JavaScript 实现，无需额外安装库。
- 使用 Google Fonts (Dancing Script & Ma Shan Zheng) 增强文字美感。
- 采用 Canvas API 实现高性能的纸屑动画效果。
